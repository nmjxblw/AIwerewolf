# AI Werewolf LLM Wiki

This directory is the Markdown knowledge layer for Track C.

Start here:

- [Track C Overview](track-c/overview.md)
- [Track C Wiki Schema](schema/track-c-wiki-schema.md)
- [Compilation Log](log.md)

## Scope

The wiki compiles raw evidence from:

- Track B published reviews
- `strategy_knowledge_docs`
- `knowledge_usage_feedback`
- evolution tournaments and retrieval-policy experiments

The wiki is not the runtime strategy source. During games, agents still retrieve from the PostgreSQL strategy pool after lifecycle and safety filtering.

## Obsidian

`docs/wiki/` can be opened as an Obsidian vault. Personal `.obsidian/` settings should stay local unless a future team decision adds shared, path-free config.
